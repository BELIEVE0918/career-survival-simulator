# 职场生存模拟器

网页版 AI 游戏化求职训练系统。

## 打开方式

直接用浏览器打开 `index.html` 即可运行。此时 AI 面试官使用本地模拟反馈。

如果要接入 Coze Agent，使用下面方式启动：

```powershell
npm install
$env:COZE_API_TOKEN="你的 Coze API Token"
$env:COZE_BOT_ID="你的 Bot ID"
npm start
```

然后打开：

```text
http://localhost:3000
```

## 第一版完整流程

1. 首页进入游戏
2. 角色创建
3. 选择目标岗位
4. 30 道岗位适配测试
5. 生成六项能力画像
6. 求职准备剧情选择
7. 求职路线复盘
8. AI 面试官 5 轮面试
9. Offer 结果判定
10. 生成职业成长报告

## 内置内容

- 12 个应届生岗位：Java 开发、数据分析、产品运营、HR 管培生、市场营销、新媒体运营、前端开发、AI 产品助理、软件测试、财务助理、电商运营、视觉设计
- 30 道岗位适配测试题：20 道通用职业题 + 10 道目标岗位题
- 6 项能力指标：专业能力、沟通能力、抗压能力、职业清晰度、人脉资源、Offer 成功率
- 5 个求职准备事件
- 5 轮 AI 面试官模拟
- 初始能力中心化归一，避免开局数值过高
- 求职准备选项包含正面收益和负面代价
- 语音输入支持开始/结束录音状态提示
- 本地保存最近训练记录和历史最高分

## Coze Agent 接入说明

当前版本已经内置 Coze 后端代理：

- 前端请求 `/api/interview`
- `server.mjs` 调用 Coze Bot
- Coze 返回面试点评
- 如果没有配置 Token 或接口失败，自动回退到本地模拟

注意：不要把 `COZE_API_TOKEN` 写进 `app.js`，只能放在后端环境变量里。
